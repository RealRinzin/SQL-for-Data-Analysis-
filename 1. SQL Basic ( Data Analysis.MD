
##### USING "HAVING" method to check the count
``` SQL
SELECT DISTINCT(SEX),COUNT(*) as Total FROM `PATIENTS` GROUP BY SEX HAVING COUNT(*) <= 2;
```

####  RANGE BETWEEN
``` SQL
SELECT * FROM EMPLOYEES WHERE (SALARY BETWEEN 60000 AND 70000) AND DEP_ID = 5 ORDER BY SALARY DESC;

 AVG with ORDER BY 
SELECT DISTINCT(DEP_ID),AVG(SALARY) AS "AvgSalary" FROM EMPLOYEES GROUP BY DEP_ID HAVING AVG(SALARY) >=60000 ORDER BY AVG(SALARY) DESC;

```
